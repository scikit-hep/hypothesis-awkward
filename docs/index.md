# hypothesis-awkward

_Hypothesis strategies for Awkward Arrays._

[hypothesis-awkward] is a collection of Hypothesis strategies for [Awkward
Array][awkward-array]. It enables property-based testing for code that uses
Awkward Arrays.

Get started with the [Guide](guide/index.md) or jump to the
[API Reference](reference/index.md).

[awkward-array]: https://awkward-array.org/
